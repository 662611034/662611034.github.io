JOOAN A6M-UのRTSPセットアップ

2026-03-12

まとめ

  • JOOAN A6M-Uという屋外用Wi-Fi監視カメラを買った
  • 設定にRTSPのURLが記載されていなかった
  • 頑張って rtsp://admin:admin123@${IP_ADDRESS}:554/live/ch0　であると突き止めた

蛇足

ブログなので、なぜこのカメラを購入し、どのような苦難と逆境を乗り越えてこの情報に辿り着いたのかについて少しぐらい書き留めたいところだが、明らかに「情報」との分量配分がおかしくなりそうなので、分かったことだけ書く。

RTSPのURL

上で書いた通り

rtsp://admin:admin123@${IP_ADDRESS}:554/live/ch0

だ。${IP_ADDRESS}にはカメラIPアドレスが入る。カメラのIPアドレスは、専用アプリ「Cam720」の設定メニューの「カメラパラメーター」から確認できるし、ルーターのネットワークマップで調べてもよい。

  • ユーザー名：admin。アプリでは見つけることができず、Google検索で分かった
  • パスワード：admin123。パスワードの方は、「カメラパラメーター」の下位メニューの「セキュリティ設定」から見つけることができた
  • サブパス: /live/ch0。Claudeに「RTSPカメラでよく使われるパス名」を調べさせて総当たりした。

Web UI

Webブラウザを使ってカメラのIPアドレスにアクセスするとWebUIがある。admin/admin123でログインすれば設定の復元やRTSP認証方法の変更などができるがLAN内で使うのであれば基本的に必要ないだろう。

英語にも対応しておらず100%中国語で、ユーザーが使うことを想定していないように見える。
